# Harmony & Rhythm

An integrated music education experience combining the 
Nashville Notation Wheel with a step sequencer — 
explore chord progressions while a beat plays underneath.

Built in vanilla HTML5. No dependencies. Works in any browser.

## What It Does
- Select a key and chord progression on the notation wheel
- Guitar chord diagrams and piano keyboard update in real time
- Drum sequencer plays underneath in the same tempo
- Shared state — key and tempo connect both tools

## Why
Music theory without rhythm is incomplete. 
This tool lets you feel harmonic relationships, 
not just see them. A student can play along with 
a beat while learning what chords belong together.
